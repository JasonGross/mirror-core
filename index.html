MirrorCore

MirrorCore aims to make building extensible & compositional reflective procedures easy in Coq. It achieves this through several means.

  1. Generic syntax provides a skeleton on which to put domain-specific terms that you want to reason about. MirrorCore provides all the plumbing for reasoning about a core lambda calculus so you can focus on your reasoning procedures.
  2. Tactic combinators provide a lightweight way to write reflective procedures very similar to the way that you write Ltac.
  3. Built-in (customizable) tactics like auto provide a quick way to write the simplest form of reflective procedures.
